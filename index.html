Publications
Memory Forcing: Spatio-Temporal Memory for Consistent Scene Generation on Minecraft
Junchao Huang, Xinting Hu, Bohan Han, Shaoshuai Shi, Zhuotao Tian, Tianyu He, Li Jiang
Under Review, 2025
pdf / project page / code (coming soon)

We introduce Memory Forcing, a training framework that enables video diffusion models to balance exploration and revisit consistency for autoregressive video generation in Minecraft using spatio-temporal memory mechanisms.
