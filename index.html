Publications
LIVE: Long-horizon Interactive Video World Modelling
Junchao Huang, Ziyang Ye, Xinting Hu, Tianyu He, Guiyu Zhang, Shaoshuai Shi, Jiang Bian, Li Jiang
International Conference on Machine Learning (ICML), 2026
pdf / project page / code (coming soon)

We introduce LIVE, a novel training framework that enforces bounded error accumulation for long-horizon video world modeling. Through a cycle-consistency constraint, LIVE performs forward rollout followed by reverse generation to reconstruct the initial state, enabling efficient autoregressive diffusion training.

Memory Forcing: Spatio-Temporal Memory for Consistent Scene Generation on Minecraft
Junchao Huang, Xinting Hu, Bohan Han, Shaoshuai Shi, Zhuotao Tian, Tianyu He, Li Jiang
Under Review, 2025
pdf / project page / code (coming soon)

We introduce Memory Forcing, a training framework that enables video diffusion models to balance exploration and revisit consistency for autoregressive video generation in Minecraft using spatio-temporal memory mechanisms.

Edit360: 2D Image Edits to 3D Assets from Any Angle
Junchao Huang, Xinting Hu, Shaoshuai Shi, Zhuotao Tian, Li Jiang
IEEE International Conference on Computer Vision (ICCV), 2025
Highlight (2.3% acceptance rate)
pdf / project page / code

A tuning-free framework that extends 2D modifications to multi-view consistent 3D editing, enabling user-specific editing from arbitrary viewpoints while ensuring structural coherence across all views.
